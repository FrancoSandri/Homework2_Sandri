# Homework2_Sandri

### Ejercicio 1

## Comandos para compilar y ejecutar el código en C++

1. Cambiar el directorio:
   ```sh
   cd Ejercicio_1
   ```

2. Compilar el código:
   ```sh
   g++ ejercicio1.cpp -o ejercicio1
   ```
2. Ejecutar el programa:
   ```sh
   ./ejercicio1
   ```
